# SmartBrief

> Stay informed. Stay balanced. Stay sane.

SmartBrief is a self-hosted personal news app that aggregates sources across topics you care about and delivers them as a calm, balanced daily briefing — without the anxiety spiral.

---

## The Problem

Most news apps are optimised for engagement, not wellbeing. They surface the most alarming headlines, repeat the same story from ten sources, and never tell you when to stop. A few minutes on a mainstream news feed often leaves you feeling worse than when you started.

## What SmartBrief Does Differently

**Aggregates, don't duplicate.** When five outlets cover the same story, SmartBrief shows you one unified brief with AI-generated bullet points that cite each source — like Perplexity for news, but yours.

**Balances good and bad.** An AI sentiment filter actively manages the ratio of positive, neutral, and negative stories in your feed. You control the mix. The default is 40% uplifting · 40% neutral · 20% in-focus.

**Respects your time.** Each session is a finite briefing (15 articles by default), not an infinite scroll. When the briefing is done, it's done.

**Stays on topic.** Four topic areas — Technology & AI, Science & Health, Business & Finance, World & Politics — drawn from curated English and German sources you choose.

**Is yours, not theirs.** No algorithm serving an advertiser's goals. No tracking. Self-hosted on your own server, accessible privately from your phone, tablet, or desktop.

---

## Features

- **AI story briefs** — Multi-source summaries generated by Claude, cited by outlet
- **Sentiment balance** — Configurable mood presets (Balanced / Constructive / Hard News)
- **Article detail view** — Tap any story for 3–5 AI bullet points before visiting the source
- **Source control** — Enable or disable individual sources including German outlets (Spiegel, Zeit, Tagesschau, Heise, t3n, Handelsblatt, Süddeutsche)
- **Bias transparency** — Each source is labelled by editorial leaning
- **Session mode** — Finite briefings with a progress bar; no doomscrolling
- **Bright Spots tab** — A dedicated feed of positive and uplifting stories
- **Saved articles** — Bookmark stories to read later
- **PWA** — Install to your iPhone or iPad home screen for a native app feel
- **Password-protected** — Private by default; your news, for your eyes only

---

## Tech Stack

- **Frontend/Backend:** Next.js 14 (App Router) + TypeScript
- **Database:** SQLite via Prisma (zero external dependencies)
- **AI:** Anthropic Claude (Haiku for sentiment & bullets, Sonnet for story briefs)
- **News sources:** RSS feeds — no API keys required beyond Anthropic
- **Deployment:** Docker + Nginx Proxy Manager on a self-hosted VPS

## Quick Start

```bash
# Clone and configure
git clone https://github.com/kevbu/smartbrief.git
cd smartbrief
cp .env.local.example .env
# Edit .env: set ANTHROPIC_API_KEY and SMARTBRIEF_PASSWORD

# Run
docker compose up -d
```

Access at `http://localhost:3000` or configure a domain via your reverse proxy.
